# OpenGL Fairy World 🧚‍♀️


## Overview

This project implements a photorealistic 3D fairy world using modern OpenGL. The goal of the project is to demonstrate real-time rendering techniques such as lighting, textures, animations, and camera control.

The scene represents a magical fairy environment containing multiple 3D objects, animated elements, and environmental effects.

## Technologies Used

The project was developed using OpenGL, GLFW, GLM, stb_image and tinyobjloader.

## Camera and Scene Interaction

The user can move through the scene using keyboard controls and rotate the camera using the mouse. The scene supports transformations such as translation, rotation and scaling. The environment can also be presented through animated camera movement.

## Lighting System

The scene includes multiple light sources such as ambient light, local lights placed in the environment and spot lights. Lighting calculations simulate realistic illumination for the objects in the scene.

## Rendering Modes

The scene can be visualized using different rendering modes including solid rendering, wireframe mode, polygon mode and smooth shading.

## Texture Mapping and Materials

Objects in the scene use texture mapping and material definitions to create a realistic appearance. Detailed textures are applied to multiple objects and interact with the lighting system.

## Shadows

The project demonstrates shadow generation, allowing objects to cast realistic shadows on the environment.

## Animations

Several elements in the scene are animated. These include the animation of the fairy wings as well as other environmental animations that make the world feel more dynamic.

## Environment Effects

The fairy world contains environmental effects such as wind simulation, rain and other dynamic scene components that increase the realism and immersion of the environment.

## Scene Description

The scene represents a fantasy fairy world containing multiple objects such as trees, mushrooms, flowers and magical structures. An animated fairy character is included in the environment. The focus of the scene is to create a magical atmosphere through detailed models, lighting and textures.

## Controls

W, A, S and D keys are used to move the camera in the scene. The mouse is used to rotate the camera and explore the environment.

## Requirements

To run the project you need Visual Studio, OpenGL, GLFW and GLM.
